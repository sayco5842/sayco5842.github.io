# 단위테스트

## 무엇을 테스트할 것인지
* Right 결과가 올바른가?
* B 경계조건은 맞는가?
* I 역 관계를 검사할 수 있는가?
* C 다른 수단을 활용하여 교차 검사할 수 있는가?
* E 오류 조건을 강제로 일어나게 할 수 있는가?
* P 성능 조건은 기준에 부합하는가?

    public void testOrderStatusCancel() throws Exception {
        # 준비
